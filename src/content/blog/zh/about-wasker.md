# 关于 Wasker the Vue

Wasker Vue 是一个专为开发者和设计师打造的现代作品集与博客主题。它基于 **Vue 3** 构建，采用了 **Vite** 作为构建工具，旨在提供极致的性能和开发体验。

## 核心特性

- **Vue 3 & Composition API**: 利用 Vue 3 的最新特性，代码组织更清晰，逻辑复用更简单。
- **响应式设计**: 无论是移动端还是桌面端，都能提供流畅的浏览体验。
- **3D 交互图谱**: 集成了 Force-Directed Graph，以独特的方式展示你的知识体系或项目关联。
- **内容为王**: 优化的排版和 Markdown 支持，让你的文章和项目展示更具吸引力。
- **国际化友好**: 内置中英文切换支持，轻松触达全球受众。

## 为什么选择 Wasker Vue？

在这个数字时代，一个独特且专业的个人网站是展示自我的最佳窗口。Wasker Vue 不仅仅是一个模板，它是一个可以随着你成长的平台。它可以是你的技术博客、设计作品集，或者是你探索 Web 技术的实验场。
